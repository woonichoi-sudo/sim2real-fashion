Text Generation with the Gemini API

The Gemini API provides comprehensive guidance on using Google's Gemini generateContent API for text generation and multimodal AI applications.

Basic Text Generation

Python

from google import genai

client = genai.Client()

response = client.models.generate_content(
    model="gemini-3.5-flash",
    contents="How does AI work?"
)
print(response.text)

JavaScript

import { GoogleGenAI } from "@google/genai";

const ai = new GoogleGenAI({});

const response = await ai.models.generateContent({
    model: "gemini-3.5-flash",
    contents: "How does AI work?"
});
console.log(response.text);

REST

POST https://generativelanguage.googleapis.com/v1beta/models/gemini-3.5-flash:generateContent

{
  "contents": [{"parts": [{"text": "How does AI work?"}]}]
}

Thinking with Gemini

Models can reason before responding, with configurable thinking levels that affect cost, latency, and intelligence trade-offs. Gemini models often have thinking enabled by default.

Available thinking levels: low, medium, high

from google import genai
from google.genai import types

client = genai.Client()

response = client.models.generate_content(
    model="gemini-3.5-flash",
    contents="Solve this step by step: What is 15% of 847?",
    config=types.GenerateContentConfig(
        thinking_config=types.ThinkingConfig(thinking_level="low")
    ),
)
print(response.text)

System Instructions and Other Configurations

Python

response = client.models.generate_content(
    model="gemini-3.5-flash",
    config=types.GenerateContentConfig(
        system_instruction="You are a cat. Your name is Neko."),
    contents="Hello there"
)
print(response.text)

REST

{
  "system_instruction": {
    "parts": [{"text": "You are a cat. Your name is Neko."}]
  },
  "contents": [{"parts": [{"text": "Hello there"}]}]
}

참고: systemInstruction은 텍스트 전용으로 사용하며, 이미지 파트는 포함하지 않습니다. (프로젝트 정리 — 공식 명시 미확인) 공식 문서의 예시는 텍스트 파트({"text": ...})만 사용하며, 이미지 등 비텍스트 파트 사용을 명시적으로 금지하는 문구는 확인되지 않았습니다.

Generation Configuration Parameters

The GenerateContentConfig object supports the following parameters:

ParameterDescription
temperatureControls randomness. Default 1.0 for Gemini 3 models. Changing below 1.0 "may lead to unexpected behavior."
topPNucleus sampling parameter
topKTop-k sampling parameter
stopSequencesSequences where generation halts
responseMimeTypeOutput format (e.g., application/json)
thinking_configThinking level configuration
system_instructionSystem-level instructions for model behavior
config = types.GenerateContentConfig(
    temperature=0.1,
    top_p=0.95,
    top_k=40,
    stop_sequences=["END"],
    response_mime_type="application/json"
)

Multimodal Inputs

Image Input (Python)

from PIL import Image
from google import genai

client = genai.Client()
image = Image.open("/path/to/organ.png")

response = client.models.generate_content(
    model="gemini-3.5-flash",
    contents=[image, "Tell me about this instrument"]
)
print(response.text)

Supported media types: images, documents, video, and audio.

Streaming Responses

Python Streaming

response = client.models.generate_content_stream(
    model="gemini-3.5-flash",
    contents=["Explain how AI works"]
)
for chunk in response:
    print(chunk.text, end="")

JavaScript Streaming

const response = await ai.models.generateContentStream({
    model: "gemini-3.5-flash",
    contents: "Explain how AI works",
});

for await (const chunk of response) {
    console.log(chunk.text);
}

REST endpoint: Uses alt=sse parameter and streamGenerateContent endpoint.

Multi-turn Conversations (Chat)

Important note: "Chat functionality is only implemented as part of the SDKs. Behind the scenes, it still uses the generateContent API. For multi-turn conversations, the full conversation history is sent to the model."

Python Chat

client = genai.Client()
chat = client.chats.create(model="gemini-3.5-flash")

response = chat.send_message("I have 2 dogs in my house.")
print(response.text)

response = chat.send_message("How many paws are in my house?")
print(response.text)

# Access conversation history
for message in chat.get_history():
    print(f'role - {message.role}', end=": ")
    print(message.parts[0].text)

Chat with Streaming

response = chat.send_message_stream("I have 2 dogs in my house.")
for chunk in response:
    print(chunk.text, end="")
