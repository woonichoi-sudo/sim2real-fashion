Media Resolution for Gemini API

The media_resolution parameter controls how the Gemini API processes media inputs by determining the maximum tokens allocated for each item.

Per-Part Media Resolution (Gemini 3 Only)

"Gemini 3 allows you to set media resolution for individual media objects within your request, offering fine-grained optimisation of token usage."

This feature enables mixing resolution levels in a single request and overrides global configuration for specific parts.

Python Example (Per-Part)

from google import genai
from google.genai import types

client = genai.Client(
  http_options={
      'api_version': 'v1alpha',
  }
)

with open('path/to/image1.jpg', 'rb') as f:
    image_bytes_1 = f.read()

image_part_high = types.Part.from_bytes(
    data=image_bytes_1,
    mime_type='image/jpeg',
    media_resolution=types.MediaResolution.MEDIA_RESOLUTION_HIGH
)

model_name = 'gemini-3.1-pro-preview'

response = client.models.generate_content(
    model=model_name,
    contents=["Describe these images:", image_part_high]
)
print(response.text)

JavaScript Example (Per-Part)

import { GoogleGenAI, MediaResolution, Part } from '@google/genai';
import * as fs from 'fs';
import { Buffer } from 'buffer';

const ai = new GoogleGenAI({ httpOptions: { apiVersion: 'v1alpha' } });

function fileToGenerativePart(path, mimeType, mediaResolution) {
    return {
        inlineData: { data: Buffer.from(fs.readFileSync(path)).toString('base64'), mimeType },
        mediaResolution: { 'level': mediaResolution }
    };
}

async function run() {
    const imagePartHigh = fileToGenerativePart('img.png', 'image/png', Part.MediaResolutionLevel.MEDIA_RESOLUTION_HIGH);
    const model_name = 'gemini-3.1-pro-preview';
    const response = await ai.models.generateContent({
        model: model_name,
        contents: ['Describe these images:', imagePartHigh]
    });
    console.log(response.text);
}
run();

REST Example (Per-Part)

IMAGE_PATH="path/to/image.jpg"
BASE64_IMAGE1=$(base64 -w 0 "$IMAGE_PATH")
MODEL_ID="gemini-3.1-pro-preview"

echo '{
    "contents": [{
      "parts": [
        {"text": "Describe these images:"},
        {
          "inline_data": {
            "mime_type": "image/jpeg",
            "data": "'"$BASE64_IMAGE1"'",
          },
          "media_resolution": {"level": "MEDIA_RESOLUTION_HIGH"}
        }
      ]
    }]
  }' > request.json

curl -s -X POST \

  -H "x-goog-api-key: $GEMINI_API_KEY" \
  -H "Content-Type: application/json" \
  -d @request.json

Global Media Resolution

This approach sets default resolution for all media parts using GenerationConfig, supported by all multimodal models.

Python Example (Global)

from google import genai
from google.genai import types

client = genai.Client()

with open('image.jpg', 'rb') as f:
    image_bytes = f.read()
image_part = types.Part.from_bytes(data=image_bytes, mime_type='image/jpeg')

config = types.GenerateContentConfig(
    media_resolution=types.MediaResolution.MEDIA_RESOLUTION_HIGH
)

response = client.models.generate_content(
    model='gemini-3.5-flash',
    contents=["Describe this image:", image_part],
    config=config
)
print(response.text)

Available Resolution Values

  • MEDIA_RESOLUTION_UNSPECIFIED: Default setting with varying token counts
  • MEDIA_RESOLUTION_LOW: Lower tokens, faster processing, reduced cost, less detail
  • MEDIA_RESOLUTION_MEDIUM: Balances detail, cost, and latency
  • MEDIA_RESOLUTION_HIGH: Higher tokens, more detail, increased latency/cost
  • MEDIA_RESOLUTION_ULTRA_HIGH: Highest tokens (per-part only), required for specialized uses like computer use

Choosing the Right Resolution

  • Default: Balanced quality, latency, cost for typical use cases
  • LOW: Cost and speed prioritized; fine detail unnecessary
  • MEDIUM/HIGH: Intricate details required; complex visual analysis, charts, dense documents
  • ULTRA_HIGH: Specific scenarios like computer use; per-part only

Version Compatibility

  • The MediaResolution enum is available for all models supporting media
  • Token counts differ between Gemini 3 and earlier versions
  • Per-part settings are exclusive to Gemini 3 models
