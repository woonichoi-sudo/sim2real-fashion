Google Gemini API - Token Counting (countTokens)

Overview

The models.countTokens method runs the model's tokenizer on the input Content and returns the token count. This allows developers to calculate how many tokens a model will consume for given input before making an API call.

API Endpoint

POST https://generativelanguage.googleapis.com/v1beta/{model=models/*}:countTokens

Path Parameters

  • model (string, required): Model resource name. Format: models/{model}

Request Body Fields

FieldTypeDescription
contents[]object (Content)Optional. Input prompt for the model
generateContentRequestobject (GenerateContentRequest)Optional. Contains system instructions and function declarations

Response Body Fields

FieldTypeDescription
totalTokensintegerTotal number of tokens the model tokenized
cachedContentTokenCountintegerToken count from cached content portions
promptTokensDetails[]object (ModalityTokenCount)List of processed modalities in the request input
cacheTokensDetails[]object (ModalityTokenCount)List of processed modalities in cached content

SDK Method Names

  • Python: client.models.count_tokens()
  • Node.js: ai.models.countTokens()
  • Go: client.Models.CountTokens()
  • Java: model.countTokens()

Code Examples

Python - Text

from google import genai

client = genai.Client()
prompt = "The quick brown fox jumps over the lazy dog."
total_tokens = client.models.count_tokens(
    model="gemini-3.5-flash", contents=prompt
)

Node.js - Text

const ai = new GoogleGenAI({ apiKey: process.env.GEMINI_API_KEY });
const countTokensResponse = await ai.models.countTokens({
  model: "gemini-3.5-flash",
  contents: prompt,
});

Go - Text

countResp, err := client.Models.CountTokens(ctx, "gemini-3.5-flash", contents, nil)

Supported Input Content Types

  • Text prompts
  • Chat conversations with history
  • Images (inline media)
  • Video files
  • PDF documents
  • Cached content
  • System instructions
  • Tool/function definitions

Important Notes

  • Token counts for images are "independent of display or file size."
  • The guide recommends consulting the comprehensive "Token counting guide" for detailed information about how different media types are tokenized.
