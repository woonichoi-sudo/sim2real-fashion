Google Gemini API - Embeddings Reference

Overview

The Gemini API offers embeddings services that convert text into numerical vectors for clustering, similarity measurement, and information retrieval tasks.

API Methods

1. models.embedContent (Single Embedding)

Endpoint:

POST https://generativelanguage.googleapis.com/v1beta/{model=models/*}:embedContent

Path Parameter: model (required) - format: models/{model}

Request Body Fields:

  • content (required): Content object to embed
  • taskType (optional, deprecated): TaskType enum value
  • title (optional, deprecated): Text title
  • outputDimensionality (optional, deprecated): Reduce output embedding dimensions
  • embedContentConfig (optional): EmbedContentConfig object

Response: EmbedContentResponse object

2. models.batchEmbedContents (Batch Embedding)

Endpoint:

POST https://generativelanguage.googleapis.com/v1beta/{model=models/*}:batchEmbedContents

Purpose: Processes multiple embedding requests simultaneously, accepting a batch of strings and returning embeddings in the same order as input requests.

Request Body:

  • requests[] (required): Array of EmbedContentRequest objects

Response Fields:

  • embeddings[]: Array of ContentEmbedding objects
  • usageMetadata: EmbeddingUsageMetadata object

3. models.asyncBatchEmbedContent (Asynchronous Batch)

Endpoint:

POST https://generativelanguage.googleapis.com/v1beta/{batch.model=models/*}:asyncBatchEmbedContent

Purpose: Enqueues batch embedding requests for asynchronous processing, useful for large-scale operations.

Data Structures

EmbedContentConfig

{
  "title": "string",
  "taskType": "enum",
  "autoTruncate": "boolean",
  "outputDimensionality": "integer",
  "documentOcr": "boolean",
  "audioTrackExtraction": "boolean"
}

TaskType Enum Values

  • TASK_TYPE_UNSPECIFIED

  • RETRIEVAL_DOCUMENT
  • SEMANTIC_SIMILARITY
  • CLASSIFICATION
  • CLUSTERING
  • QUESTION_ANSWERING
  • FACT_VERIFICATION
  • CODE_RETRIEVAL_QUERY

ContentEmbedding

{
  "values": [number],
  "shape": [integer]
}

EmbeddingUsageMetadata

{
  "promptTokenCount": "integer",
  "promptTokenDetails": [ModalityTokenCount]
}

EmbedContentBatchStats

  • requestCount: Total number of requests in the batch
  • successfulRequestCount: Number of successfully processed requests
  • failedRequestCount: Number of failed requests
  • pendingRequestCount: Number of requests still being processed

Code Examples

Python

from google import genai
from google.genai import types

client = genai.Client()
text = "Hello World!"
result = client.models.embed_content(
    model="gemini-embedding-001",
    contents=text,
    config=types.EmbedContentConfig(output_dimensionality=10),
)
print(result.embeddings)

Node.js

const ai = new GoogleGenAI({ apiKey: process.env.GEMINI_API_KEY });
const text = "Hello World!";
const result = await ai.models.embedContent({
  model: "gemini-embedding-001",
  contents: text,
  config: { outputDimensionality: 10 },
});
console.log(result.embeddings);

Go

ctx := context.Background()
client, err := genai.NewClient(ctx, &genai.ClientConfig{
    APIKey:  os.Getenv("GEMINI_API_KEY"),
    Backend: genai.BackendGeminiAPI,
})

text := "Hello World!"
outputDim := int32(10)
contents := []*genai.Content{
    genai.NewContentFromText(text, genai.RoleUser),
}
result, err := client.Models.EmbedContent(ctx, "gemini-embedding-001",
    contents, &genai.EmbedContentConfig{
        OutputDimensionality: &outputDim,
})

embeddings, _ := json.MarshalIndent(result.Embeddings, "", "  ")
fmt.Println(string(embeddings))

Shell/cURL

curl "https://generativelanguage.googleapis.com/v1beta/models/gemini-embedding-001:embedContent" \
-H "x-goog-api-key: $GEMINI_API_KEY" \
-H 'Content-Type: application/json' \
-d '{"model": "models/gemini-embedding-001",

Important Usage Notes

  • Users must retain responsibility for uploaded content and confirm they possess necessary rights.
  • The service prohibits content violating intellectual property or privacy rights.
  • Subject to Google's Prohibited Use Policy and Terms of Service.
  • autoTruncate: Enables auto-truncation for oversized content.
  • documentOcr: Enables document OCR feature.
  • audioTrackExtraction: Enables audio extraction feature.
