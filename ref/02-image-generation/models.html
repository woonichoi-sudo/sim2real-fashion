Google Gemini API Models Overview

Model Version Name Patterns

  • Stable: Points to a specific stable model; stable models usually do not change.
  • Preview: Preview models; may be production-ready with billing enabled; limited rate limits; minimum 2-week deprecation notice.
  • Latest: Points to the latest release of a specific model variant; changes with each new release (hot-swapped).
  • Experimental: Experimental models; not suitable for production; more limited rate limits.
