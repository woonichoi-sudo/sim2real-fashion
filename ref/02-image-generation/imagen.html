Generate Images Using Imagen

Google's Imagen is described as a "high-fidelity image generation model, capable of generating realistic and high quality images from text prompts." The service generates 1–4 images per request with automatic SynthID watermarking.

Generate Images Using the Imagen Models

Python Example

from google import genai
from google.genai import types

client = genai.Client()

response = client.models.generate_images(
    model='imagen-4.0-generate-001',
    prompt='Robot holding a red skateboard',
    config=types.GenerateImagesConfig(
        number_of_images=4,
    )
)

for generated_image in response.generated_images:
    image = generated_image.image
    image.show()

JavaScript Example

import { GoogleGenAI } from "@google/genai";

const ai = new GoogleGenAI({ apiKey: process.env.GEMINI_API_KEY });

const response = await ai.models.generateImages({
  model: "imagen-4.0-generate-001",
  prompt: "Robot holding a red skateboard",
  config: { numberOfImages: 4 },
});

for (const image of response.generatedImages) {
  console.log(image.image.imageBytes);
}

Go Example

Imagen Configuration Parameters

ParameterTypeDefaultDescription
numberOfImagesinteger4Number of images to generate (1–4)
imageSizestring"1K"Image size: "1K" or "2K" (Standard/Ultra only)
aspectRatiostring"1:1"Aspect ratio: "1:1", "3:4", "4:3", "9:16", "16:9"
personGenerationstring"allow_adult"Person generation policy: "dont_allow", "allow_adult", "allow_all"

Note: "allow_all" is not permitted in EU, UK, CH, and MENA regions.

Imagen Prompt Guide

Prompt Writing Basics

Effective prompts combine three elements:

  1. Subject — the main focus of the image
  2. Context/Background — environment and setting
  3. Style — artistic style, medium, or aesthetic

Maximum prompt length: 480 tokens

Photography Prompts

Subject TypeRecommended Lens
Portraits (people)24–35mm
Food / objects60–105mm macro
Sports / animals100–400mm telephoto
Astronomy / landscapes10–24mm wide-angle

Style Categories

  • Photography (photorealistic images, portraits, objects, motion, wide-angle)
  • Illustration and art
  • Shapes and materials
  • Historical art references
  • Image quality modifiers

Generate Text in Images

Imagen can generate text within images. Best results with:

  • Maximum 25 characters of text per prompt
  • Clear, short phrases for menus, infographics, diagrams

Prompt Parameterization

Use variables in prompts to create systematic variations. For example, vary subject, style, or setting programmatically.

Advanced Prompt Writing Techniques

  • Iterate and refine prompts based on results
  • Specify lighting conditions (e.g., "golden hour lighting", "studio lighting")
  • Specify camera angles (e.g., "bird's eye view", "close-up")
  • Use artistic style references (e.g., "oil painting", "watercolor", "digital art")

Supported Languages

"Imagen supports English only prompts at this time."

Important Notes

  • All generated images include a SynthID watermark.
  • Maximum prompt length is 480 tokens.
  • "allow_all" for personGeneration is not available in EU, UK, CH, and MENA regions.
  • Text generation in images: limit to 25 characters for best results.
  • imageSize parameter ("1K" or "2K") is only available for Standard and Ultra variants.
