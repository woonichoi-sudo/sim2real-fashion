Gemini API: Generating Content

This page documents the Gemini API's content generation capabilities, specifically the models.generateContent and models.streamGenerateContent methods.

Method: models.generateContent

Endpoint

POST /v1beta/{model=models/*}:generateContent

Path Parameters

ParameterTypeDescription
modelstringRequired. The name of the Model to use for generating the completion. Format: models/{model}

Request Body Fields

FieldTypeRequiredDescription
contents[]Content objectRequiredThe content of the current conversation with the model
tools[]Tool objectOptionalA list of Tools the Model may use to generate the next response
toolConfigToolConfig objectOptionalTool configuration for any Tool specified in the request
safetySettings[]SafetySetting objectOptionalA list of unique SafetySetting instances for blocking unsafe content. There should not be more than one setting for each SafetyCategory type.
systemInstructionContent objectOptionalDeveloper set system instruction(s). Currently, text only.
generationConfigGenerationConfig objectOptionalConfiguration options for model generation and outputs
cachedContentstringOptionalThe name of the content cached to use as context. Format: cachedContents/{cachedContent}
serviceTierServiceTier enumOptionalThe service tier of the request
storebooleanOptionalConfigures the logging behavior for a given request

GenerationConfig Parameters

stop_sequencesarraySet of character sequences that will stop output generation
max_output_tokensintegerMaximum number of tokens to include in a response candidate
temperaturefloatControls the randomness of the output

topKintegerMaximum number of tokens to consider when sampling
seedintegerSeed used in decoding. If not set, the request uses a randomly generated seed. (공식 페이지 확인 불가 — 값 미기재)
responseMimeTypestringMIME type of the generated candidate text. Supported values: text/plain (default), application/json (JSON response), text/x.enum (enum value). (text/x.enum 공식 페이지 확인 불가 — 값 미기재)

Method: models.streamGenerateContent

Endpoint

POST /v1beta/{model=models/*}:streamGenerateContent

Uses Server-Sent Events (SSE) for streaming responses. Same request body as generateContent.

Response: GenerateContentResponse

FieldTypeDescription
candidates[]Candidate objectCandidate responses from the model
promptFeedbackPromptFeedback objectReturns the prompt's feedback related to the content filters
usageMetadataUsageMetadata objectMetadata on the generation requests' token usage
modelVersionstringThe model version used to generate the response
responseIdstringresponseId is used to identify each response
modelStatusModelStatus objectThe current model status of this model

PromptFeedback

FieldTypeDescription
blockReasonBlockReason enumIf set, the prompt was blocked and no candidates are returned
safetyRatings[]SafetyRating objectRatings for safety of the prompt

UsageMetadata

FieldTypeDescription

toolUsePromptTokenCountintegerNumber of tokens in tool-use prompt(s)

Code Examples

Python - Text Generation

from google import genai

client = genai.Client()
response = client.models.generate_content(
    model="gemini-3.5-flash",
    contents="Write a story about a magic backpack."
)
print(response.text)

Python - Streaming

for chunk in client.models.generate_content_stream(
    model="gemini-3.5-flash",
    contents="Write a long story."
):
    print(chunk.text, end="")

Python - Chat (Multi-turn)

chat = client.chats.create(model="gemini-3.5-flash")
response = chat.send_message("Hello")
print(response.text)

Python - Image Input

import PIL.Image
image = PIL.Image.open("image.jpg")
response = client.models.generate_content(
    model="gemini-3.5-flash",
    contents=["What is in this image?", image]
)

Python - Audio Input

audio_file = client.files.upload(file=media / "sample.mp3")
response = client.models.generate_content(
    model="gemini-3.5-flash",
    contents=["Describe this audio.", audio_file]
)

Python - JSON Mode

from google.genai import types

response = client.models.generate_content(
    model="gemini-3.5-flash",
    contents="List 3 colors in JSON",
    config=types.GenerateContentConfig(
        response_mime_type="application/json"
    )
)

Python - Safety Settings

from google.genai import types

response = client.models.generate_content(
    model="gemini-3.5-flash",
    contents="...",
    config=types.GenerateContentConfig(
        safety_settings=[
            types.SafetySetting(
                category="HARM_CATEGORY_HATE_SPEECH",
                threshold="BLOCK_LOW_AND_ABOVE"
            )
        ]
    )
)

Python - System Instruction

response = client.models.generate_content(
    model="gemini-3.5-flash",
    contents="Hello",
    config=types.GenerateContentConfig(
        system_instruction="You are a helpful assistant."
    )
)

Python - Context Caching

cache = client.caches.create(
    model="gemini-3.5-flash",
    contents=[...],
    config=types.CreateCachedContentConfig(ttl="600s")
)
response = client.models.generate_content(
    model="gemini-3.5-flash",
    contents="Question about cached content",
    config=types.GenerateContentConfig(cached_content=cache.name)
)

    APIKey: os.Getenv("GEMINI_API_KEY"),

    ctx,

    genai.Text("Write a story about a magic backpack."),

)

Important Notes

  • Input capabilities differ between models, including tuned models.
  • There should not be more than one SafetySetting for each SafetyCategory type.
  • The API will block any contents and responses that fail to meet the thresholds set by safety settings.
  • For video processing: after upload, poll until file state is ACTIVE before using in generateContent.
  • cachedContents must use format: cachedContents/{cachedContent}
  • Safety ratings and content filtering are reported in GenerateContentResponse.prompt_feedback and for each candidate.
  • SDK default API version is v1beta; use v1 for stable production environments.
