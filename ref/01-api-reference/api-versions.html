Gemini API Versions Explained

Last updated: May 28, 2026

Available Versions

v1 (Stable)

"Stable version of the API. Features in the stable version are fully-supported over the lifetime of the major version."

Non-breaking changes may be introduced without changing the major version. Breaking changes would only occur in a new major version with a deprecation period. Recommended for production environments.

v1beta (Preview / Beta)

"This version includes early features that may be under development and are subject to breaking changes."

Warning: "If you require stability in your production environment and cannot risk breaking changes, you should not use this version in production."

SDK default: v1beta. Includes newer and experimental features not yet in v1.

Configure API Version in an SDK

Python

from google import genai

client = genai.Client(
    api_key="YOUR_API_KEY",
    http_options={"api_version": "v1"}  # Use stable v1
)

JavaScript

import { GoogleGenAI } from "@google/genai";

const ai = new GoogleGenAI({
  apiKey: "YOUR_API_KEY",
  httpOptions: { apiVersion: "v1" }  // Use stable v1
});

REST

# v1 (stable)
curl "https://generativelanguage.googleapis.com/v1/models/gemini-2.0-flash:generateContent?key=$GEMINI_API_KEY" \
  -H 'Content-Type: application/json' \
  -d '{"contents": [{"parts": [{"text": "Hello"}]}]}'

# v1beta (includes experimental features)
curl "https://generativelanguage.googleapis.com/v1beta/models/gemini-2.0-flash:generateContent?key=$GEMINI_API_KEY" \
  -H 'Content-Type: application/json' \
  -d '{"contents": [{"parts": [{"text": "Hello"}]}]}'
