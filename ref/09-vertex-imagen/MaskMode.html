MaskMode | Vertex AI Imagen | Google Cloud

MaskMode는 Imagen 이미지 편집 API에서 마스크 레이어 생성 방식을 지정하는 열거형(enum)입니다. maskImageConfig.maskMode 필드에 사용합니다.

열거형 값 (Enum Values)

값 설명
MASK_MODE_DEFAULT 기본 마스크 모드 (Default mask mode). 명시적으로 마스크 모드를 지정하지 않을 때의 기본값입니다.
MASK_MODE_USER_PROVIDED 개발자가 직접 공급한 마스크 이미지를 사용합니다 (User provided mask). referenceImages 배열 내 REFERENCE_TYPE_MASK 항목의 referenceImage.bytesBase64Encoded 필드에 마스크 이미지를 제공해야 합니다. 마스크는 편집 대상 영역을 표시하는 흑백(black and white) 이미지입니다.
권장 dilation 값: 0.01 (삽입/제거 용도), 0.03 (아웃페인팅 용도), 0.00 (제품 배경 교체)
MASK_MODE_BACKGROUND 배경 세그멘테이션을 사용하여 자동으로 마스크를 생성합니다 (Background mask). "Automatically generates a mask using background segmentation." 별도 마스크 이미지 제공 불필요. 배경 영역을 편집할 때 사용.
MASK_MODE_FOREGROUND 전경 세그멘테이션을 사용하여 자동으로 마스크를 생성합니다 (Foreground mask). "Automatically generates a mask using foreground segmentation." 별도 마스크 이미지 제공 불필요. 주로 전경 객체 삽입(insertion) 및 제거(removal)에 사용.
권장 dilation 값: 0.1
MASK_MODE_SEMANTIC 시맨틱 세그멘테이션(semantic segmentation)을 사용하여 자동으로 마스크를 생성합니다 (Semantic mask). "Automatically generates a mask using semantic segmentation based on the segmentation classes you specify in the maskImageConfig.maskClasses array." maskClasses 배열에 세그멘테이션 클래스 ID를 지정해야 합니다. 예시: [175, 176] → 자전거(bicycle) 및 자동차(car) 클래스.

maskImageConfig 객체 구조

{
  "maskMode": "MASK_MODE_DEFAULT" | "MASK_MODE_USER_PROVIDED" | "MASK_MODE_BACKGROUND" | "MASK_MODE_FOREGROUND" | "MASK_MODE_SEMANTIC",
  "dilation": float,         // 0~1 사이 값, 마스크를 이미지 너비 비율만큼 팽창(확장)
  "maskClasses": [integer]   // MASK_MODE_SEMANTIC 사용 시에만 필요한 클래스 ID 배열
}

필드 설명

  • maskMode (string, enum): 마스크 생성 방식 지정 (위 열거형 값 참조)
  • dilation (float, 0~1): 마스크를 이미지 너비의 지정 비율만큼 팽창시킵니다. "The percentage of image width to dilate this mask by. A value of 0.01 is recommended to compensate for imperfect input masks." 삽입/제거: 0.01 권장, 아웃페인팅: 0.03 권장(0.0은 확장 지점에 뚜렷한 경계/흰색 테두리 발생 가능), 제품 이미지: 0.00 권장.
  • maskClasses (integer[]): MASK_MODE_SEMANTIC에서만 사용. 세그멘테이션 클래스 ID 배열. 예: [175, 176] (자전거, 자동차).

referenceImages 내 마스크 참조 구조

{
  "referenceType": "REFERENCE_TYPE_MASK",
  "referenceId": 2,
  "referenceImage": {
    "bytesBase64Encoded": "BASE64_ENCODED_MASK_IMAGE"
    // MASK_MODE_USER_PROVIDED 사용 시에만 필요
  },
  "maskImageConfig": {
    "maskMode": "MASK_MODE_USER_PROVIDED",
    "dilation": 0.01
  }
}

사용 예시

사용자 제공 마스크 (MASK_MODE_USER_PROVIDED)

{
  "referenceType": "REFERENCE_TYPE_MASK",
  "referenceId": 2,
  "referenceImage": {
    "bytesBase64Encoded": "B64_MASK_IMAGE"
  },
  "maskImageConfig": {
    "maskMode": "MASK_MODE_USER_PROVIDED",
    "dilation": 0.01
  }
}

자동 전경 감지 (MASK_MODE_FOREGROUND)

{
  "referenceType": "REFERENCE_TYPE_MASK",
  "referenceId": 2,
  "maskImageConfig": {
    "maskMode": "MASK_MODE_FOREGROUND",
    "dilation": 0.1
  }
}

시맨틱 세그멘테이션 (MASK_MODE_SEMANTIC)

{
  "referenceType": "REFERENCE_TYPE_MASK",
  "referenceId": 2,
  "maskImageConfig": {
    "maskMode": "MASK_MODE_SEMANTIC",
    "maskClasses": [175, 176],
    "dilation": 0.01
  }
}

사용 가능한 EditMode 조합

EditMode 사용 가능한 MaskMode 권장 dilation
EDIT_MODE_INPAINT_INSERTION USER_PROVIDED, FOREGROUND, BACKGROUND, SEMANTIC 0.01 (USER_PROVIDED), 0.1 (자동)
EDIT_MODE_INPAINT_REMOVAL USER_PROVIDED, FOREGROUND, BACKGROUND, SEMANTIC 0.01 (USER_PROVIDED)
EDIT_MODE_OUTPAINT USER_PROVIDED 0.03
EDIT_MODE_BGSWAP USER_PROVIDED, BACKGROUND, FOREGROUND 0.00 (제품), 0.03 (일반)
