Imagen API customization | Vertex AI | Google Cloud

Imagen 커스터마이제이션 API는 참조 이미지를 기반으로 특정 피사체(subject)나 스타일(style)을 유지하면서 새로운 이미지를 생성할 수 있게 해줍니다.

지원 모델

  • imagen-3.0-capability-001

HTTP 엔드포인트

POST https://LOCATION-aiplatform.googleapis.com/v1/projects/PROJECT_ID/locations/LOCATION/publishers/google/models/imagen-3.0-capability-001:predict

경로 파라미터

  • PROJECT_ID: Google Cloud 프로젝트 ID
  • LOCATION: 지역 엔드포인트 (예: us-central1, europe-west2, asia-northeast3)

1. 피사체 커스터마이제이션 (Subject Customization)

특정 인물, 제품, 동물 등의 피사체를 유지하면서 새로운 장면이나 스타일의 이미지를 생성합니다.

요청 본문 구조

{
  "instances": [
    {
      "prompt": "watercolor portrait of woman [1] in pose of [2]",
      "referenceImages": [
        {
          "referenceType": "REFERENCE_TYPE_SUBJECT",
          "referenceId": 1,
          "referenceImage": {
            "bytesBase64Encoded": "BASE64_PERSON_IMAGE"
          },
          "subjectImageConfig": {
            "subjectDescription": "a headshot of a woman",
            "subjectType": "SUBJECT_TYPE_PERSON"
          }
        },
        {
          "referenceType": "REFERENCE_TYPE_CONTROL",
          "referenceId": 2,
          "referenceImage": {
            "bytesBase64Encoded": "BASE64_FACEMESH_IMAGE"
          },
          "controlImageConfig": {
            "controlType": "CONTROL_TYPE_FACE_MESH",
            "enableControlImageComputation": true
          }
        }
      ]
    }
  ],
  "parameters": {
    "sampleCount": 1
  }
}

referenceImages 필드 상세 (피사체 커스터마이제이션)

필드타입필수설명
referenceType string (enum) 필수
  • REFERENCE_TYPE_SUBJECT: 피사체 참조 이미지
  • REFERENCE_TYPE_CONTROL: 제어(포즈/표정) 참조 이미지
referenceId integer 필수 참조 이미지 식별자. 프롬프트에서 [referenceId] 형식으로 참조 (예: [1], [2])
referenceImage.bytesBase64Encoded string 조건부 Base64 인코딩된 이미지 데이터 (또는 gcsUri 사용)
referenceImage.gcsUri string 조건부 Google Cloud Storage URI (예: gs://bucket/path). bytesBase64Encoded 대신 사용 가능.
subjectImageConfig.subjectDescription string 선택 참조 피사체에 대한 텍스트 설명 (예: "a headshot of a woman")
subjectImageConfig.subjectType string (enum) 필수 피사체 유형:
  • SUBJECT_TYPE_PERSON: 인물
  • SUBJECT_TYPE_PRODUCT: 제품/물체
  • SUBJECT_TYPE_ANIMAL: 동물
controlImageConfig.controlType string (enum) 선택 CONTROL_TYPE_FACE_MESH: 얼굴 메쉬 제어 (현재 지원되는 유일한 타입)
controlImageConfig.enableControlImageComputation boolean 선택 true로 설정 시 자동으로 제어 구조 감지

피사체 유형별 요구 사항

subjectType사용 사례참고 사항
SUBJECT_TYPE_PERSON 스타일 변형, 포즈 변형 얼굴이 중앙에 위치, 프레임의 50% 이상 차지, 정면 촬영, 가려짐 없어야 함
SUBJECT_TYPE_PRODUCT 객체 배치, 색상 변형 각도 보존은 성공률 낮음. 명확한 객체 가시성, 일관된 조명 권장
SUBJECT_TYPE_ANIMAL 동물 이미지 정체성 보존에 적합하지 않음

Python 코드 예시 (피사체 커스터마이제이션)

from google import genai
from google.genai.types import (
    ControlReferenceConfig,
    ControlReferenceImage,
    EditImageConfig,
    Image,
    SubjectReferenceConfig,
    SubjectReferenceImage,
)

# 환경 변수 설정:
# export GOOGLE_CLOUD_PROJECT=PROJECT_ID
# export GOOGLE_CLOUD_LOCATION=us-central1
# export GOOGLE_GENAI_USE_VERTEXAI=True

client = genai.Client()

subject_ref = SubjectReferenceImage(
    reference_id=1,
    reference_image=Image(gcs_uri="gs://bucket/person.png"),
    config=SubjectReferenceConfig(
        subject_description="a headshot of a woman",
        subject_type="SUBJECT_TYPE_PERSON",
    ),
)

control_ref = ControlReferenceImage(
    reference_id=2,
    reference_image=Image(gcs_uri="gs://bucket/facemesh.png"),
    config=ControlReferenceConfig(control_type="CONTROL_TYPE_FACE_MESH"),
)

image = client.models.edit_image(
    model="imagen-3.0-capability-001",
    prompt="watercolor portrait of woman [1] in pose of [2]",
    reference_images=[subject_ref, control_ref],
    config=EditImageConfig(
        edit_mode="EDIT_MODE_DEFAULT",
        number_of_images=1,
        safety_filter_level="BLOCK_MEDIUM_AND_ABOVE",
    ),
)

2. 스타일 커스터마이제이션 (Style Customization)

참조 이미지의 스타일(예: 네온사인, 모자이크, 수채화)을 적용하여 새로운 이미지를 생성합니다.

요청 본문 구조

{
  "instances": [
    {
      "prompt": "Generate an image in neon sign style [1] based on the following caption: a sign saying have a great day",
      "referenceImages": [
        {
          "referenceType": "REFERENCE_TYPE_STYLE",
          "referenceId": 1,
          "referenceImage": {
            "bytesBase64Encoded": "BASE64_ENCODED_IMAGE_BYTES"
          },
          "styleImageConfig": {
            "styleDescription": "neon sign"
          }
        }
      ]
    }
  ],
  "parameters": {
    "sampleCount": 2
  }
}

referenceImages 필드 상세 (스타일 커스터마이제이션)

필드타입필수설명
referenceType string (enum) 필수 REFERENCE_TYPE_STYLE: 스타일 참조 이미지
referenceId integer (1~4) 필수 참조 이미지 식별자. 프롬프트에서 [referenceId] 형식으로 참조
referenceImage.bytesBase64Encoded string 조건부 Base64 인코딩된 이미지 데이터
referenceImage.gcsUri string 조건부 Google Cloud Storage URI. bytesBase64Encoded 대신 사용 가능.
styleImageConfig.styleDescription string 선택 참조 스타일에 대한 텍스트 설명 (예: "neon sign", "mosaic style"). 생성 품질 향상에 도움.

parameters 필드

필드타입범위기본값설명
sampleCount integer 1~4 4 생성할 이미지 수

프롬프트 패턴 예시

사용 사례프롬프트 템플릿예시
객체 스타일 변환 Generate an image in STYLE_DESCRIPTION [1] based on the following caption: IMAGE_DESCRIPTION "Generate an image in neon sign style [1] based on the following caption: a sign saying have a great day"
인물 스타일화 (얼굴 메쉬 없이) Create an image about SUBJECT_DESCRIPTION [1] to match the description: a portrait of SUBJECT_DESCRIPTION [1] ${PROMPT} -
인물 스타일화 (얼굴 메쉬 사용) Create an image about SUBJECT_DESCRIPTION [1] in the pose of the CONTROL_IMAGE [2] to match the description: a portrait of SUBJECT_DESCRIPTION [1] ${PROMPT} -

Python 코드 예시 (스타일 커스터마이제이션)

from google import genai
from google.genai.types import (
    EditImageConfig,
    Image,
    StyleReferenceConfig,
    StyleReferenceImage,
)

client = genai.Client()

# GCS URI를 사용한 스타일 참조 이미지
style_reference_image = StyleReferenceImage(
    reference_id=1,
    reference_image=Image(gcs_uri="gs://cloud-samples-data/generative-ai/image/neon.png"),
    config=StyleReferenceConfig(style_description="neon sign"),
)

image = client.models.edit_image(
    model="imagen-3.0-capability-001",
    prompt="generate an image of a neon sign [1] with the words: have a great day",
    reference_images=[style_reference_image],
    config=EditImageConfig(
        edit_mode="EDIT_MODE_DEFAULT",
        number_of_images=1,
        safety_filter_level="BLOCK_MEDIUM_AND_ABOVE",
        person_generation="ALLOW_ADULT",
        output_gcs_uri="gs://your-bucket/your-prefix",
    ),
)

print(image.generated_images[0].image.gcs_uri)

공통 제한 사항

  • 요청당 최대 4개의 참조 이미지 사용 가능
  • 이미지는 Base64 인코딩 또는 GCS URI 형식으로 제공
  • 스타일 설명(styleDescription)은 선택 사항이지만 생성 품질 향상에 도움
  • 출력 형식: PNG (MIME 타입: image/png)
  • 제어 참조 이미지(controlImageConfig)는 요청당 하나만 지원
