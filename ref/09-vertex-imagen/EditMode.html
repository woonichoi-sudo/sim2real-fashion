EditMode | Vertex AI Imagen | Google Cloud

EditMode는 Imagen 이미지 편집 API에서 수행할 편집 작업의 유형을 지정하는 열거형(enum)입니다. parameters.editMode 필드에 사용합니다.

열거형 값 (Enum Values)

값 설명 주요 사용 케이스
EDIT_MODE_INPAINT_INSERTION 마스크로 지정된 영역에 새로운 콘텐츠(객체, 요소)를 삽입합니다. 텍스트 프롬프트로 삽입할 내용을 설명합니다. 이미지에 객체 추가, 빈 공간 채우기, 특정 영역 내용 교체. 예: "A plate of cookies"를 프롬프트로 지정한 영역에 쿠키 접시 삽입
EDIT_MODE_INPAINT_REMOVAL 마스크로 지정된 영역의 콘텐츠(객체)를 제거하고, 배경에 맞게 자연스럽게 채웁니다. 자동 마스킹 사용 시 프롬프트를 비워두는 것이 권장됩니다. 원치 않는 객체 제거, 사람/물체 지우기, 배경 복원. 예: 이미지에서 자전거 제거
EDIT_MODE_OUTPAINT 이미지의 경계를 확장하여 더 큰 캔버스로 콘텐츠를 늘립니다. 마스크는 확장할 영역(현재 이미지 외부)을 지정합니다. outpaintingConfig와 함께 사용하여 블렌딩 설정 가능. 이미지 크기 확장, 파노라마 생성, 배경 확장. 예: 세로 이미지를 가로 이미지로 확장
EDIT_MODE_BGSWAP 이미지의 배경을 교체하면서 전경(전면 객체)은 보존합니다. 프롬프트로 새 배경을 설명합니다. 제품 사진 배경 교체, 인물 사진 배경 변경, 장면 재설정. 예: 흰색 배경의 제품 이미지를 공항 창문 앞 장면으로 교체

parameters 내 editMode 사용 구조

{
  "parameters": {
    "editMode": "EDIT_MODE_INPAINT_INSERTION",
    "editConfig": {
      "baseSteps": 35
    },
    "sampleCount": 4
  }
}

editConfig 객체

editMode와 함께 사용하는 편집 설정 객체입니다.

필드 타입 설명
baseSteps integer 베이스 모델 샘플링 스텝 수. 높을수록 품질 향상(지연 시간 증가).
  • 삽입(INPAINT_INSERTION): 35 시작, 최대 75까지 조정
  • 제거(INPAINT_REMOVAL): 12 시작, 최대 75까지 조정
  • 아웃페인팅(OUTPAINT): 35 시작
  • 배경 교체(BGSWAP): 75 권장 (제품 이미지)
outpaintingConfig object EDIT_MODE_OUTPAINT 전용 설정.
  • blendingMode: "alpha-blending" 사용
  • blendingFactor: 0.01 권장 (블렌딩 경계 처리)

모드별 권장 설정 요약

EditMode 권장 baseSteps 프롬프트 권장 MaskMode
EDIT_MODE_INPAINT_INSERTION 35 (최대 75) 삽입할 객체 설명 (필수) USER_PROVIDED, FOREGROUND, BACKGROUND, SEMANTIC
EDIT_MODE_INPAINT_REMOVAL 12 (최대 75) 빈 문자열 권장 (자동 마스크 사용 시) USER_PROVIDED, FOREGROUND, BACKGROUND, SEMANTIC
EDIT_MODE_OUTPAINT 35 빈 문자열 또는 확장 영역 설명 USER_PROVIDED
EDIT_MODE_BGSWAP 75 (제품) 새 배경 설명 (필수) USER_PROVIDED, BACKGROUND, FOREGROUND, PEOPLE
