Imagen API - Edit images | Vertex AI | Google Cloud

Imagen 이미지 편집 API를 사용하면 마스크 기반 인페인팅(inpainting), 아웃페인팅(outpainting), 배경 교체(background swap) 등의 편집 작업을 수행할 수 있습니다.

지원 모델

  • imagen-3.0-capability-001 — 현재 편집 API의 기본 지원 모델

HTTP 엔드포인트

POST https://LOCATION-aiplatform.googleapis.com/v1/projects/PROJECT_ID/locations/LOCATION/publishers/google/models/imagen-3.0-capability-001:predict

요청 본문 (Request Body)

{
  "instances": [
    {
      "prompt": "TEXT_PROMPT",
      "referenceImages": [
        {
          "referenceType": "REFERENCE_TYPE_RAW",
          "referenceId": 1,
          "referenceImage": {
            "bytesBase64Encoded": "BASE64_BASE_IMAGE"
          }
        },
        {
          "referenceType": "REFERENCE_TYPE_MASK",
          "referenceId": 2,
          "referenceImage": {
            "bytesBase64Encoded": "BASE64_MASK_IMAGE"  // USER_PROVIDED 일 때만 필요
          },
          "maskImageConfig": {
            "maskMode": "MASK_MODE_USER_PROVIDED",
            "dilation": 0.01
          }
        }
      ]
    }
  ],
  "parameters": {
    "editMode": "EDIT_MODE_INPAINT_INSERTION",
    "editConfig": {
      "baseSteps": 35
    },
    "sampleCount": 4
  }
}

instances 파라미터

필드타입필수설명
prompt string 조건부 편집 내용을 설명하는 텍스트 프롬프트. 삽입 작업 시 필수. 제거 작업 시 자동 마스크 사용 경우 빈 문자열 권장.
referenceImages array 필수 편집에 사용할 참조 이미지 목록. 원본 이미지(RAW)와 마스크 이미지(MASK)를 포함.

referenceImages 항목 구조

필드타입설명
referenceType string (enum) 참조 이미지 유형:
  • REFERENCE_TYPE_RAW: 편집할 원본 이미지
  • REFERENCE_TYPE_MASK: 편집 영역을 지정하는 마스크 이미지
referenceId integer 참조 이미지를 식별하는 고유 ID (예: 1, 2)
referenceImage.bytesBase64Encoded string Base64 인코딩된 이미지 데이터. 최대 10 MB.
maskImageConfig object REFERENCE_TYPE_MASK 항목에서 마스크 생성 방식 설정. maskMode, dilation, maskClasses 포함.

maskImageConfig 구조

필드타입설명
maskMode string (enum) 마스크 생성 방식:
  • MASK_MODE_DEFAULT: 기본 마스크 모드
  • MASK_MODE_USER_PROVIDED: 직접 제공한 마스크 이미지 사용
  • MASK_MODE_FOREGROUND: 자동 전경 감지
  • MASK_MODE_BACKGROUND: 자동 배경 감지
  • MASK_MODE_SEMANTIC: 시맨틱 세그멘테이션 (maskClasses 지정 필요)
dilation float (0~1) 마스크를 이미지 너비의 지정 비율만큼 팽창(확장). 마스크 경계 부정확성 보완.
  • USER_PROVIDED (삽입/제거): 0.01 권장
  • 자동 감지 (삽입): 0.1 권장
  • 아웃페인팅: 0.03 권장
  • 제품 배경 교체: 0.00 권장
maskClasses integer[] MASK_MODE_SEMANTIC 전용. 시맨틱 세그멘테이션 클래스 ID 배열. 예: [175, 176] (자전거, 자동차)

parameters 파라미터

필드타입필수설명
editMode string (enum) 필수 편집 작업 유형:
  • EDIT_MODE_INPAINT_INSERTION: 마스크 영역에 객체 삽입
  • EDIT_MODE_INPAINT_REMOVAL: 마스크 영역의 객체 제거
  • EDIT_MODE_OUTPAINT: 이미지 경계 확장
  • EDIT_MODE_BGSWAP: 배경 교체 (전경 보존)
editConfig object 선택 편집 설정 객체 (아래 editConfig 구조 참조)
sampleCount integer (1~4) 선택 생성할 이미지 변형 수. 기본값: 4
seed integer 선택 결정론적 출력을 위한 시드 값
guidanceScale float 선택 프롬프트 준수 강도 조절
negativePrompt string 선택 생성 결과에서 제외할 내용 설명
safetyFilterLevel string 선택 콘텐츠 필터링 수준 조절
personGeneration string 선택 인물 생성 방식 제어
[프로젝트 노트 — 실사화 작업용]
  • negativePrompt 예시: anime, cartoon, 2d, blurry (비실사 요소 제외)
  • guidanceScale: 실사화에는 21+ 권장 (프롬프트 반영 강도 높임)

editConfig 구조

필드타입설명
baseSteps integer 샘플링 스텝 수. 높을수록 품질 향상(지연 시간 증가). 권장값:
  • 삽입(INPAINT_INSERTION): 35 시작, 최대 75
  • 제거(INPAINT_REMOVAL): 12 시작, 최대 75
  • 아웃페인팅(OUTPAINT): 35 시작
  • 배경 교체(BGSWAP) 제품: 75 권장
outpaintingConfig object EDIT_MODE_OUTPAINT 전용.
  • blendingMode: "alpha-blending"
  • blendingFactor: 0.01 권장

코드 예시

객체 삽입 (EDIT_MODE_INPAINT_INSERTION) - Python

from google import genai
from google.genai.types import (
    RawReferenceImage,
    MaskReferenceImage,
    MaskReferenceConfig,
    EditImageConfig,
)

# 환경 변수 설정 필요:
# export GOOGLE_CLOUD_PROJECT=PROJECT_ID
# export GOOGLE_CLOUD_LOCATION=us-central1
# export GOOGLE_GENAI_USE_VERTEXAI=True

client = genai.Client()

raw_ref = RawReferenceImage(
    reference_image=Image.from_file(location="base_image.png"),
    reference_id=0,
)
mask_ref = MaskReferenceImage(
    reference_id=1,
    reference_image=Image.from_file(location="mask_image.png"),
    config=MaskReferenceConfig(
        mask_mode="MASK_MODE_USER_PROVIDED",
        mask_dilation=0.01,
    ),
)

image = client.models.edit_image(
    model="imagen-3.0-capability-001",
    prompt="A plate of cookies",
    reference_images=[raw_ref, mask_ref],
    config=EditImageConfig(
        edit_mode="EDIT_MODE_INPAINT_INSERTION",
    ),
)
image.generated_images[0].image.save("output.png")

아웃페인팅 (EDIT_MODE_OUTPAINT) - Python

from google import genai
from google.genai.types import (
    RawReferenceImage,
    MaskReferenceImage,
    MaskReferenceConfig,
    EditImageConfig,
)

client = genai.Client()

raw_ref = RawReferenceImage(
    reference_image=Image.from_file(location="living_room.png"),
    reference_id=0,
)
mask_ref = MaskReferenceImage(
    reference_id=1,
    reference_image=Image.from_file(location="living_room_mask.png"),
    config=MaskReferenceConfig(
        mask_mode="MASK_MODE_USER_PROVIDED",
        mask_dilation=0.03,
    ),
)

image = client.models.edit_image(
    model="imagen-3.0-capability-001",
    prompt="A chandelier hanging from the ceiling",
    reference_images=[raw_ref, mask_ref],
    config=EditImageConfig(edit_mode="EDIT_MODE_OUTPAINT"),
)
image.generated_images[0].image.save("output-image.png")

제한 사항 및 참고 사항

  • 이미지 최대 크기: 10 MB (Base64 인코딩 기준)
  • 마스크 이미지는 원본 이미지와 동일한 해상도여야 합니다.
  • 출력 해상도는 원본 참조 이미지의 크기와 동일합니다.
  • 아웃페인팅 시 원본 이미지의 200% 이상 확장 시 왜곡이 발생할 수 있습니다.
  • 제거 시 마스크 인접 영역의 작은 객체들도 함께 제거될 수 있습니다.
  • 마스크 외 영역의 변경은 보장되지 않습니다.
