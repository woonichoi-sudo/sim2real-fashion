Imagen API | Vertex AI | Google Cloud

Imagen API는 텍스트 프롬프트로부터 이미지를 생성하는 API입니다. Vertex AI의 Imagen 모델을 사용하여 고품질 이미지를 생성할 수 있습니다.

지원 모델

현재 지원 생성 모델

  • imagen-4.0-generate-001
  • imagen-4.0-fast-generate-001
  • imagen-4.0-ultra-generate-001
  • imagen-3.0-generate-002
  • imagen-3.0-generate-001
  • imagen-3.0-fast-generate-001

사용 중단 예정 모델 (2026년 6월 30일까지 gemini-2.5-flash-image로 마이그레이션 필요)

  • imagegeneration@002
  • imagegeneration@003
  • imagegeneration@004
  • imagegeneration@005
  • imagegeneration@006
  • imagetext@001
  • imagen-3.0-capability-001
  • imagen-3.0-capability-002

HTTP 엔드포인트

POST https://REGION-aiplatform.googleapis.com/v1/projects/PROJECT_ID/locations/REGION/publishers/google/models/MODEL_VERSION:predict

요청 본문 (Request Body)

기본 구조

{
  "instances": [
    {
      "prompt": "TEXT_PROMPT"
    }
  ],
  "parameters": {
    "sampleCount": IMAGE_COUNT
  }
}

전체 파라미터 예시

{
  "instances": [
    {
      "prompt": "A dog reading a newspaper"
    }
  ],
  "parameters": {
    "sampleCount": 4,
    "aspectRatio": "1:1",
    "addWatermark": true,
    "enhancePrompt": true,
    "seed": 12345,
    "safetySetting": "block_medium_and_above",
    "personGeneration": "allow_adult",
    "outputOptions": {
      "mimeType": "image/png",
      "compressionQuality": 75
    },
    "storageUri": "gs://my-bucket/output/"
  }
}

instances 파라미터

필드타입필수설명
prompt string 필수 생성할 이미지를 설명하는 텍스트 프롬프트

parameters 파라미터

필드타입기본값설명
sampleCount integer (1~4) 4 생성할 이미지 수
aspectRatio string (enum) "1:1" 이미지 비율:
  • 1:1 — 정사각형
  • 3:4 — 세로 (풀스크린)
  • 4:3 — 가로 (풀스크린)
  • 16:9 — 와이드스크린
  • 9:16 — 세로 (와이드)
addWatermark boolean true SynthID 디지털 워터마크 추가 여부. seed 파라미터 사용 시 false로 설정 필요.
enhancePrompt boolean true LLM 기반 프롬프트 자동 개선(rewriting) 활성화 여부
seed integer (1~2147483647) 없음 결정론적 출력을 위한 시드 값. 사용 시 addWatermark를 false로 설정해야 함.
safetySetting string (enum) "block_medium_and_above" 안전 필터 수준:
  • block_low_and_above — 낮은 수준 이상 차단 (가장 엄격)
  • block_medium_and_above — 중간 수준 이상 차단
  • block_only_high — 높은 수준만 차단 (가장 완화)
personGeneration string (enum) "allow_adult" 인물/얼굴 생성 제어:
  • allow_all — 모든 연령 허용
  • allow_adult — 성인만 허용
  • dont_allow — 인물 생성 금지
outputOptions object 없음 출력 형식 설정 (아래 outputOptions 구조 참조)
storageUri string 없음 Cloud Storage 출력 경로. 예: gs://my-bucket/output/
includeRaiReason boolean true 안전 필터 사유 코드 포함 여부
includeSafetyAttributes boolean false RAI 안전 점수 포함 여부 (아래 안전 속성 목록 참조)

outputOptions 구조

필드타입설명
mimeType string (enum) 이미지 파일 형식:
  • image/jpeg
  • image/png
  • image/gif
  • image/webp
  • image/bmp
  • image/tiff
  • image/vnd.microsoft.icon
compressionQuality integer (0~100) JPEG 압축 품질. 기본값: 75

응답 본문 (Response Body)

{
  "predictions": [
    {
      "bytesBase64Encoded": "BASE64_IMG_BYTES",
      "mimeType": "image/png",
      "prompt": "ENHANCED_PROMPT_TEXT"
    }
  ]
}

응답 필드

필드타입설명
predictions array 생성된 이미지 객체 배열
bytesBase64Encoded string Base64 인코딩된 이미지 데이터
mimeType string 이미지 MIME 타입
prompt string LLM이 개선한 프롬프트 텍스트 (모델이 지원하는 경우)

코드 예시

Python (google-genai SDK)

# 설치
# pip install --upgrade google-genai

# 환경 변수
# export GOOGLE_CLOUD_PROJECT=YOUR_PROJECT
# export GOOGLE_CLOUD_LOCATION=global
# export GOOGLE_GENAI_USE_VERTEXAI=True

from google import genai
from google.genai.types import GenerateImagesConfig

client = genai.Client()

image = client.models.generate_images(
    model="imagen-4.0-generate-001",
    prompt="A dog reading a newspaper",
    config=GenerateImagesConfig(
        image_size="2K",
    ),
)

image.generated_images[0].image.save("output_file.png")
print(f"Created output image using {len(image.generated_images[0].image.image_bytes)} bytes")

안전 속성 (includeSafetyAttributes: true 시 반환)

다음 카테고리별 RAI 안전 점수가 반올림된 값으로 제공됩니다:

  • Death, Harm & Tragedy
  • Firearms & Weapons
  • Hate
  • Health
  • Illicit Drugs
  • Politics
  • Porn
  • Religion & Belief
  • Toxic
  • Violence
  • Vulgarity
  • War & Conflict

워터마크 관련 참고 사항

  • 기본적으로 SynthID 디지털 워터마크가 생성된 이미지에 자동으로 추가됩니다.
  • 콘솔에서는 워터마크를 비활성화할 수 없습니다.
  • API에서 "addWatermark": false로 설정하여 비활성화 가능합니다.
  • seed 파라미터 사용 시 반드시 "addWatermark": false로 설정해야 합니다.
