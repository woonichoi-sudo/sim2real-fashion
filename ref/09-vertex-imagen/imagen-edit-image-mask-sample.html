Edit image content using a mask with Imagen v.002

Overview

This Google Cloud documentation page provides guidance on using the Imagen model for mask-based image editing through the Vertex AI platform.

Purpose

The sample demonstrates how to use Imagen to perform targeted edits on images by specifying a mask area and providing a text prompt describing desired modifications.

Setup Requirements

  • Python environment with Vertex AI client libraries
  • Application Default Credentials for authentication
  • Input image file, mask image file, and desired output location

Code Sample (Python)

The provided Python code demonstrates the implementation:

import vertexai
from vertexai.preview.vision_models import Image, ImageGenerationModel

# TODO(developer): Update and un-comment below lines
# PROJECT_ID = "your-project-id"

# output_file = "output-image.png"
# prompt = "" # The text prompt describing what you want to see.

vertexai.init(project=PROJECT_ID, location="us-central1")

model = ImageGenerationModel.from_pretrained("imagegeneration@002")
base_img = Image.load_from_file(location=input_file)
mask_img = Image.load_from_file(location=mask_file)

images = model.edit_image(
    base_image=base_img,
    mask=mask_img,
    prompt=prompt,
    # Optional parameters
    seed=1,
    # Controls the strength of the prompt.
    # -- 0-9 (low strength), 10-20 (medium strength), 21+ (high strength)
    guidance_scale=21,
    number_of_images=1,
)

images[0].save(location=output_file, include_generation_parameters=False)

# Optional. View the edited image in a notebook.
# images[0].show()

print(f"Created output image using {len(images[0]._image_bytes)} bytes")
# Example response:
# Created output image using 971614 bytes

Key Parameters

  • base_image: The base image to edit
  • mask: The mask image specifying which area to edit
  • prompt: Text prompt describing desired edits
  • seed: For reproducibility
  • guidance_scale: Controls prompt strength:
    • 0–9: Low strength
    • 10–20: Medium strength
    • 21+: High strength
  • number_of_images: Quantity of output images
  • include_generation_parameters: Whether to include generation parameters in saved output (False in example)
